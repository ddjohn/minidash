# minidash

## install

Install meteor:
> $ curl https://install.meteor.com/ | sh

Clone the repo:
> $ git clone http://github.com/ddjohn/minidash

Build 'minidash':
> $ cd minidash
> $ meteor update
> $ meteor npm install

## run

Run 'mini dash':
> $ meteor
